# CALCULADORA DE IMC

Projeto em ReactJS + Typescript desenvolvendo uma calculadora de IMC.

Projeto desenvolvido durante o curso [B7Web](https://b7web.com.br)

### Instalação:
- `yarn install`

### Para rodar o projeto:
- `yarn dev`
